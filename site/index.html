    ESSE É O FUTURO

    Nós utilizaremos o arduino
    UNO e o sensor de umidade (MTS)
    também conhecido como Moisture Sensor
    para a captação de dados. Logo após
    iremos lançar esses dados
    no banco. E por último mas não
    menos importante, vamos usar
    os dados coletados para
    transformá-los em uma informação
    intuitiva para o usuario atraves
    da nossa dashboard.
